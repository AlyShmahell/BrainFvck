# BrainMachina
an Interpreter for Brainf*ck (Turing tar-pit Language), implemented using Bison, Flex and GCC.

## Installation
  - ### on Ubuntu (18.04):
    ```sh
    sudo pip3 install git+https://github.com/AlyShmahell/BrainMachina
    ```
## Running
#### with stdin
```sh
BrainMachina
```
#### with file input 
```sh
BrainMachina <file_name>
```
